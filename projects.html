EU claim validator

Ulepszona wersja wcześniejszego projektu. Na podstawie otrzymanych danych wartości odżywczych, generuje rapot o RDI i dostępnych oświadczeniach żywieniowych zgodnych z prawem UE.

Tech: HTML, CSS + Flexbox i czysty JS + 'html2pdf'.

Supreme Bank

Aplikacja pseudo bankowa z trzema sekcjami: kalkulatorem pożyczek, kalkulatorem funduszu oszczędnościowego i kolektora API ze spółkami giełdowymi o najwyższym wzroście.

Tech: HTML, CSS + Flexbox i czysty JS z Axios

Nutrition Converter

Walidator wartości odżywczych oparty na rozporządzeniach unijnych (1169/2011 i 1924/2006). Pozwala przekształcić dane na wartość RDI i wykazuje dostępne dla produktu oświadczenia żywieniowe.

Tech: HTML, Bootstrap 4, czysty JS z 'html2pdf' i odrobnia jQuerry i CSS

Production Panel

Prosty panel zarządzania produkcją. Oparty na strukturze kart, pozwala dodawać nowe sprawy i nadawać im priorytet za pomocą podwójnego kliknięcia. W pełni responsywny, zawiera wykresy Googla.

Tech: HTML, CSS + Flexbox i czysty JS + Google Charts

Github Rank

Prosta aplikacja podbierająca dane o najpopularniejszych repozytoriach na githubie (w oparciu o ilość gwiazd). Możliwy jest wybór spośród 11 popularnych języków programowania. Po pobraniu danych generowany jest wykres porównujący popularność repozytoriów.
Tech: HTML, Bootstrap 4 i czysty JS z Charts.js
